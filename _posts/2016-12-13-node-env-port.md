**Port**

```bash
$ PORT=1234 node app.js
```
**More permanently**

```bash
$ export PORT=1234
$ node app.js
```

**In Windows**

```bash
set PORT=1234
```

**In Windows PowerShell**

```bash
$env:PORT = 1234
```
