**Port**

```bash
$ PORT=80 node app.js
```
**More permanently**

```bash
$ export PORT=80
$ node app.js
```

**In Windows**

```bash
set PORT=80
```

**In Windows PowerShell**

```bash
$env:PORT = 80
```
